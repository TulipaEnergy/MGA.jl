NearOptimalAlternatives.jl Documentation

NearOptimalAlternatives.jl is a package for generating near optimal alternative solutions to a solved JuMP.jl optimisation problem. The alternative solutions are within a maximum specified percentage of the optimum and are as different from the optimal solution (and other alternatives) as possible. Alternatives can either be generated using mathematical optimisation or using a metaheuristic algorithm. For the latter, this package depends on Metaheuristics.jl.
